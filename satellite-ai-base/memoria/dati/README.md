# memoria/dati/

Qui l'AI salva **le serie storiche che ha effettivamente usato** in ogni comitato,
una per mese: `AAAA-MM_serie.csv`.

Non è una cartella tecnica: è **prova**. Serve a una cosa sola — che chiunque, tra un
anno, possa ripartire dal verbale di quel mese, riprendere questo file, e **rifare
esattamente gli stessi conti**. Se i numeri di un verbale non sono ricalcolabili da qui,
quel verbale è un'opinione ben scritta.

Regole:
- Fonte **pubblica e gratuita**, dichiarata nel verbale. Chiunque deve poterla rifare.
- Si salva la serie **usata**, non "la si riscarica dopo": i dati vengono ri-aggiustati
  nel tempo e la vista di domani non è quella di oggi.
- Non si cancella mai niente. I file vecchi sono la memoria, non ingombro.
